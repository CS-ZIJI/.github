## 标准工作流程


### 仓库命名规则

- 除特殊仓库，其它仓库命名应当遵守 `{Flag}-{Name}` 的形式。
- 其中，`{Flag}` 标识仓库功能，`{Name}` 标识仓库内容。
- `{Flag}` 包括：
    - T(tutorial): 笔记、教程类内容资源，例如数字逻辑设计课程笔记。
    - F(functional): 功能类仓库，例如模板库。
    - ...
